# Política de Privacidade Localizador de Qibla

Última atualização: 29 de maio de 2025

## Introdução

Obrigado por usar o aplicativo Localizador de Qibla. Esta política de privacidade explica como seus dados pessoais são coletados, usados e protegidos quando você usa nosso aplicativo.

## Coleta de Dados

### Informações de Localização
- O aplicativo usa as informações de localização do seu dispositivo para determinar a direção da Qibla
- As informações de localização são usadas apenas para calcular a direção da Qibla
- As informações de localização são processadas localmente no seu dispositivo e não são enviadas para nenhum servidor

## Uso de Dados

Os dados coletados são usados para os seguintes propósitos:
- Cálculo preciso da direção da Qibla
- Fornecer a funcionalidade básica do aplicativo

## Segurança de Dados

- Todos os dados são armazenados localmente no seu dispositivo
- Seus dados não são compartilhados com terceiros
- Nenhum dado é enviado para servidores

## Direitos do Usuário

Como usuários, vocês têm os seguintes direitos:
- Alterar permissões de localização a qualquer momento
- Excluir todos os seus dados desinstalando o aplicativo
