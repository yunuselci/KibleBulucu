# Dasar Privasi Pencari Kiblat

Kemaskini terakhir: 29 Mei 2025

## Pengenalan

Terima kasih kerana menggunakan aplikasi Pencari Kiblat. Dasar privasi ini menerangkan bagaimana data peribadi anda dikumpul, digunakan dan dilindungi semasa menggunakan aplikasi kami.

## Pengumpulan Data

### Maklumat Lokasi
- Aplikasi menggunakan maklumat lokasi peranti anda untuk menentukan arah kiblat
- Maklumat lokasi hanya digunakan untuk mengira arah kiblat
- Maklumat lokasi diproses secara tempatan pada peranti anda dan tidak dihantar ke mana-mana pelayan

## Penggunaan Data

Data yang dikumpul digunakan untuk tujuan berikut:
- Pengiraan tepat arah kiblat
- Menyediakan fungsi asas aplikasi

## Keselamatan Data

- Semua data disimpan secara tempatan pada peranti anda
- Data anda tidak dikongsi dengan pihak ketiga
- Tiada data dihantar ke pelayan

## Hak Pengguna

Sebagai pengguna, anda mempunyai hak berikut:
- Mengubah kebenaran lokasi pada bila-bila masa
- Memadam semua data anda dengan menyahpasang aplikasi
