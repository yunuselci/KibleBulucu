# Kebijakan Privasi Pencari Kiblat

Terakhir diperbarui: 29 Mei 2025

## Pendahuluan

Terima kasih telah menggunakan aplikasi Pencari Kiblat. Kebijakan privasi ini menjelaskan bagaimana data pribadi Anda dikumpulkan, digunakan, dan dilindungi saat menggunakan aplikasi kami.

## Pengumpulan Data

### Informasi Lokasi
- Aplikasi menggunakan informasi lokasi perangkat Anda untuk menentukan arah kiblat
- Informasi lokasi hanya digunakan untuk menghitung arah kiblat
- Informasi lokasi diproses secara lokal di perangkat Anda dan tidak dikirim ke server manapun

## Penggunaan Data

Data yang dikumpulkan digunakan untuk tujuan berikut:
- Perhitungan akurat arah kiblat
- Menyediakan fungsionalitas dasar aplikasi

## Keamanan Data

- Semua data disimpan secara lokal di perangkat Anda
- Data Anda tidak dibagikan dengan pihak ketiga
- Tidak ada data yang dikirim ke server

## Hak Pengguna

Sebagai pengguna, Anda memiliki hak-hak berikut:
- Mengubah izin lokasi kapan saja
- Menghapus semua data Anda dengan menghapus instalan aplikasi
